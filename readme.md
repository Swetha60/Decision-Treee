# Task 7: Decision Tree

## Project Overview

In this task, I implemented a Decision Tree model using the Titanic dataset. The objective was to understand how decision trees work by splitting data based on conditions and making predictions. I also visualized the tree structure and evaluated the model performance.

---

## Tools Used

* Python
* NumPy
* Pandas
* Scikit-learn
* Matplotlib

---

## Dataset

I used the Titanic dataset.

Selected features:

* Pclass (Passenger class)
* Sex
* Age
* Fare

Target variable:

* Survived (0 → No, 1 → Yes)

---

## Steps Performed

1. Loaded the dataset using pandas
2. Selected relevant columns for analysis
3. Handled missing values in the Age column
4. Converted categorical data (Sex) into numerical format
5. Separated features (X) and target (y)
6. Split the dataset into training and testing sets
7. Built a Decision Tree model
8. Trained the model using training data
9. Predicted results on test data
10. Evaluated performance using accuracy and precision
11. Visualized the decision tree

---

## Results

* The model achieved good accuracy on the test data
* Precision provided insight into correct positive predictions
* Tree visualization helped in understanding decision-making steps
